Documentation for ext:ns_ext_compatibility
===========================================

:Extension:    `ns_ext_compatibility <https://extensions.typo3.org/extension/ns_ext_compatibility/>`_
               for TYPO3

:Version:      |release|

:Description:
    One of the only TYPO3 extension which provides all the features like TYPO3 Extensions Compatibility Report, Extensions Statistics, System overview report, Compatible from TYPO3 4.x to 8.x. This extension will help you to check feasibility and complexity to analyze upgrade estimation and define a perfect upgrade workflow.
